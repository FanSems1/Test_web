# HTTP STATUS CODE

**Successful Response(2xx)**

    - 200 OK (GET, PATCH)
  
    - 201 Created (POST, PUT)

 **CLIENT ERROR Response (4xx)**

    - 400 Bad Req
    - 401 Unautorized
    - 403 Forbidden
    - 404 Not Found
    - 405 Method not Allowed

**Server Error Response (5xx)**

    - 500 Internal Server Error
